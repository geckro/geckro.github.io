Essentials

Operating system

Windows 11 for Windows-based operating systems, and EndeavourOS or Fedora for Linux-based operating systems.

Web browser

For Windows/Linux, I recommend Thorium for Chromium browsers, or Firefox if you don't want to support Blink.

For Android, I recommend Cromite (active fork of Bromite) or Firefox Mobile. I can't give advice on iOS as I do not use Apple stuff.

I recommend these following extensions:

  • uBlock Origin - This is the only ad/content blocker you should use. If you're not sure on what extensions to use, use this one.
  • SponsorBlock - Allows you to skip sponsored segments of videos, and more (filler segments, non music sections)
  • DeArrow - It replaces titles/thumbnails on YouTube with crowdsourced literal, non-clickbait titles/thumbnails.
  • Violentmonkey - Userscript manager, open source version of Tampermonkey and updated version of Greasemonkey.
  • Stylus - Custom CSS on websites. Useful to make dark-mode websites light-mode.
  • Unhook - Remove various annoying features of YouTube.
  • BlockTube - Block YouTube channels from appearing on YouTube.
  • uBlacklist - Blocks sites from appearing in Google search results.

Terminal emulator

Konsole for Linux, and Windows Terminal for Windows.

Multimedia

Music player

MusicBee for Windows, and Strawberry for Linux (but also available on Windows). Also consider foobar2000 on Windows. I'm not a fan of music players that are extremely simple, but Amberol is there for the GNOME users.

Video player

Use mpv. If you need a no-fuss all in one video player, just use VLC. Also consider Haruna for an mpv frontend, but I can't vouch for it.

Image editor

GIMP is good enough for what I need to do, although it really needs a revamp (and a much better name). For pixel art, I recommend Aseprite.

Digital art

Krita doesn't cost $5 trillion dollars and a sacrifice to get the software so I use it :p

I also use OpenTabletDriver for my Wacom tablet.

Other

Text editors & IDEs

VSCode. For Python and .NET I use JetBrains PyCharm and Rider respectively. If you don't have a JB license for .NET, download Visual Studio (but it is Windows only).

Password manager

Bitwarden. If you don't want to rely on Bitwarden's servers and cannot self-host, use KeePassXC.

Note taking

I use Notion. If you don't want to have your notes online, try out Obsidian.

BitTorrent clients

I rarely actually torrent nowadays, but when i need "Linux ISOs" i use qBittorrent. If you use GNOME, check out Fragments.

Office suites

LibreOffice should be fine for most people, but you can just use the Google suite, or if you really need the complicated MS stuff then just use 365. If you somehow cannot or do not want to touch MS Office but still need to have maximum compatibility with it, try OnlyOffice.

Email

I use Gmail, on a browser. I also use Google Calendar, Drive, Maps, Photos and such. For a more "private" email service, try Proton. I don't use an email client, but if you really need an email client I recommend Thunderbird as it has gotten a modern refresh recently. For a more minimal email client, try Geary.

Games

Launchers

For the most part, I just use Steam. Sometimes I end up making a Playnite library to consolidate all my launchers, but it usually gets abandoned. For Minecraft, I exclusively use Prism Launcher.

Emulators

  • 32X: RetroArch (PicoDrive)
  • 3DS: Citra (fork)
  • Dreamcast: Flycast
  • DS/i: RetroArch (melonDS) or melonDS (Standalone)
  • GameCube / Wii: Dolphin
  • Game Boy / Color: RetroArch (SameBoy or Gambatte)
  • Game Boy Advance: RetroArch (mGBA) or mGBA (Standalone)
  • Genesis/Master System/Game Gear/Sega CD: RetroArch (GenesisPlusGX)
  • N64: RetroArch (Mupen64), or Rosalie's Mupen GUI, or Parallel Launcher for ROM hacks
  • NES: RetroArch (Mesen)
  • PS1: DuckStation, PS2: PCSX2, PS3: RPCS3, PSP: PPSSPP, PSVita: Vita3K
  • Saturn: RetroArch (Beetle Saturn)
  • SNES: RetroArch (Snes9x)
  • Switch: Ryujinx
  • Wii U: Cemu
  • Xbox: Xemu, Xbox 360: Xenia or Xenia Canary
