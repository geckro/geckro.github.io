üîê Privacy & Security

Enhanced Tracking Protection

‚úÖ Strict

This does not break most websites and only helps to protect security. If you need to turn off tracking protection look at the top left of the address bar. You can turn it off per website there.

‚ùå Do Not Track

Do Not Track is not what it says. It basically sends a request to every website you visit and the website can choose to honour that request or not. And most of the time, they don't. It's just additional tracking information.

Cookies and Site Data

üîò Delete cookies and site data when Firefox is closed

Cookies are still used to track you, but if you clear cookies then you will need to log-in to every website unless you make exceptions to those websites. It's not really worth it nowadays as websites have much easier methods to track you, like fingerprinting.

Cookie Banner Reduction

‚úÖ Reduce Cookie Banners

Firefox automatically rejects cookie requests on sites that show you that "Can i use your cookies". This only helps to improve your privacy.

Logins and Passwords

üîò Ask to save logins and passwords for websites

Firefox's inbuilt password manager is secure and encrypted. Mozilla cannot see your passwords. You can learn more about this here. If you use another password manager, t urn this off

‚úÖ Show alerts about passwords for breached websites

If you use Firefox's password manager, be sure to turn on this setting. Basically, it checks known website breaches and if the date that you saved a password for your website is before the breach happened, Firefox will tell you to change your password. The breach data is provided by haveibeenpwned, a trustworthy site.

‚úÖ Use a Primary Password

If you use Firefox's password manager, be sure to turn on this setting. This makes it so you need a password to enter your logins so someone on your PC cant access it, like a family member.

History

Read

History cannot be accessed by websites. It is simply something that is stored on your pc. Only clear history if you have to hide it from someone.

Permissions

Location, Camera, Microphone, Notifications, Autoplay, Virtual Reality

All of these are set to "Always Ask" by default. If you Set these to "Block", you are opening yourself up to device fingerprinting. Keep all of these settings by default.

‚úÖ Block pop-up windows

Some websites like to open popups that are usually malicious. If theres a site that is trustworthy, you can open the popup on that site.

‚úÖ Warn you when websites try to install add-ons

Stops malware sites from installing malicious addons/extensions into your browser.

Firefox Data Collection and Use

Before you turn these off:

This is data that gets sent to Mozilla to help improve Firefox. If you use Firefox Beta/Developer/Nightly, leave it on to help Firefox. The data gets used by developers to help develop Firefox. Mozilla does not sell this data to any third-party from what i know. If you want to help Mozilla develop Firefox, leave it on. If you don't want to send data, turn it off.

Deceptive Content and Dangerous Software Protection

‚úÖ Block dangerous and deceptive content

These settings use Google Safe Browsing to block malware that Google have listed in their database. If you don't want to connect to Google's servers, leave this off. You have to choose whether you want Privacy or Security.

Certificates

‚úÖ Query OCSP responder servers to confirm the current validity of certificates

This is another Security or Privacy scenario. In order to check that a website hasn't been tampered with, Firefox will send data to the certificate authority of that website to check if the site is valid. This means that you will be pinging all CA's constantly.

HTTPS-Only Mode

‚úÖ Enable HTTPS-Only Mode in all windows

Literally nearly every site uses HTTPS, and if it doesn't you'll get a warning. HTTP is unsecure and anyone can see what you're doing on that website. With HTTPS, they can only see the domain you're on. i.e. if you're searching on google.com, they can only see you're on google.com. But if you're using the HTTP version, they can see exactly what you searched.

üîÅ Sync

If you have multiple devices then Sign up for a Firefox account. This allows you to sync your passwords, history, bookmarks and else between devices. Mozilla cannot see any of your data in your account.

‚ö†Ô∏è Advanced (about:config)

This can help you tweak everything in Firefox but heed the warning Firefox gives you, you can screw up your browser. You can access this by entering about:config in the address bar. The amount of customizations you can do to Firefox with about:config tweaks alone is gigantic and I'll only show a few different user.js files for you to look through.

  • arkenfox user.js - This is a template file that hardens Firefox for security, privacy and anti-tracking features. I will reiterate that this is a template and it is advised to read the wiki and look through user.js file itself.
  • pyllyukko user.js - Similar to arkenfox's user.js script except it seems that it goes a lot further with browser hardening. Seems overkill but it exists if you want it.
  • yokoffing Betterfox - Has tweaks to make Firefox faster, make Firefox minimal and also some privacy/security tweaks.
  • r/Firefox about:config tweaks - A bunch of general use customizations from r/Firefox.

    • SponsorBlock - This automatically skips Sponsorships on YouTube. You can also make it skip self promotion, interaction reminders, intros, non music sections and recaps.
    • Return YouTube Dislike - Returns the YouTube dislike.
    • FastForward - You know those adf.ly and ouo.io links that make you wait 5 seconds before you can continue? This extension skips those.
    • Dark Reader - If you like dark mode but some sides are pure white (like Wikipedia), this extension turns them dark.
    • Bypass Paywalls Clean - This has been removed from the Firefox addons store for "copyright" reasons but this bypasses paywalls from websites that require you to pay to see an article.
    • Skip Redirect - Some sites like to add a tracking referrer before directing you to a website. This disables that.
    • Web Archives - Allows you to view archived and cached versions of a webpage, useful when a certain site is having a mass blackout.
    • Augmented Steam - Made by IsThereAnyDeal that improves the Steam website with a bunch of useful features.

    Extensions to avoid

    • I Don't Care about Cookies - This was recently acquired by Avast which is known to be scummy. Use I still don't care about cookies, an open source continuation of the former.
    • Privacy Badger - This extension is practically useless as it "learns" to block trackers. So it doesn't block them all until its "learnt" it. uBlock Origin blocks these by default.
    • HTTPS Everywhere - Redundant with HTTPS Only Mode. This extension was removed from the store but do not bother with any other HTTPS addons.
    • Decentraleyes/LocalCDN - if you use Strict tracking protection, third party sites are already partitioned and isolated.If you care about not connecting to certain CDN domains, use a VPN or TOR instead.
    • Any other adblockers (AdBlock/AdBlock Plus) - uBlock Origin is already the best adblocker and a lot of these don't have the features uBlock Origin enables. Some of these extensions can also be malware.
    • ClearURLs - if you set-up URL tracking protection in uBlock Origin, this extension is redundant.
